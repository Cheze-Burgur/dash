# Dash
Web based Geometry Dash clone made with pure JavaScript. Made by me.

## Features:
- Six of the eight original gd gamemodes
  - Wave and Swing still WIP
- Reverse and Normal gravity
- Yellow, Magenta, Red, and Blue Pads and Orbs
  - Also includes Dash Orbs
- Fully customizable level layout inside the script
- Coins
  - Coins currently have no functionality; will be added in due time
- Movable attempt counter object and level end orb.
